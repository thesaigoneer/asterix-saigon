# Asterix Saigon

My adaptation of Aurora KDE



# Rebase

Install Fedora Kinoite or Sway first, then:

rpm-ostree rebase ostree-unverified-registry:ghcr.io/thesaigoneer/asterix-saigon:latest

systemctl reboot

rpm-ostree rebase ostree-image-signed:docker://ghcr.io/thesaigoneer/asterix-saigon:latest

--------------

This about concludes a bare-bone install of Chimera Linux. I personally added a number of KDE applications afterwards and installed and activated flatpak. But that's all up to the end user (being yourself obviously).
